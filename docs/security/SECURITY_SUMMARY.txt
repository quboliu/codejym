🎯 客户端开发指南
--------------------------------------------------------------------------------
✅ 正确方式:
  const token = await login(email, password);
  localStorage.setItem('token', token);
  fetch('/api/data', {
    headers: { 'Authorization': `Bearer ${token}` }
  });

❌ 错误方式:
  document.cookie = "token=xxx";  // 无效！
  fetch('/api/data?token=xxx');   // 不安全！
