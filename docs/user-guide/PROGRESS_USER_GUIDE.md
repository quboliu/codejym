# 用户进度管理使用指南

## 🎯 功能说明

CodeJYM 现在支持用户学习进度的实时持久化和重置功能。

## 📌 核心特性

### 1. 自动进度保存 ✅
- **何时保存**: 每输入 1.2 秒自动保存一次
- **保存内容**:
  - 当前字符位置（进度）
  - 错误次数
  - 练习用时
- **存储位置**: 云端数据库（永不下线）
- **恢复方式**: 刷新页面后自动恢复

### 2. 重置进度功能 ✅
- **位置**: 练习界面工具栏右侧
- **样式**: 红色按钮
- **作用**: 一键将当前文档进度归零

## 🚀 使用方法

### 开始练习
1. 登录 CodeJYM
2. 上传或选择代码文件
3. 点击文件开始练习
4. 系统自动创建会话并保存进度

### 查看进度
- **顶部统计栏**:
  - 进度百分比
  - 准确率百分比
  - 练习用时

- **侧边栏信息**:
  - 错误次数
  - 估算 WPM（每分钟字数）
  - Session ID（会话标识）

### 重置进度
1. 在练习界面，点击工具栏中的 **"重置进度"** 按钮（红色）
2. 确认对话框中点击 **"确定"**
3. 进度将立即归零：
   - 进度 → 0%
   - 错误次数 → 0
   - 练习用时 → 00:00

⚠️ **注意**: 重置操作不可撤销，请谨慎使用！

## 💡 常见问题

### Q: 刷新页面后进度会丢失吗？
**A**: 不会！进度已自动保存到数据库，刷新后会自动恢复。

### Q: 关闭浏览器再打开，进度还在吗？
**A**: 在！只要使用同一个账户登录，进度永远保存在云端。

### Q: 重置后可以恢复吗？
**A**: 不可以。重置操作会永久清除当前文档的进度，请谨慎操作。

### Q: 自动保存会消耗很多流量吗？
**A**: 不会。每次保存只传输几个数字，数据量极小。

### Q: 可以手动保存进度吗？
**A**: 不需要。系统每 1.2 秒自动保存，无需手动操作。

## 📊 数据说明

### 存储的数据
- **进度位置**: 你在文档中输入到第几个字符
- **错误次数**: 按错键的次数
- **练习用时**: 从开始到现在的总时间（秒）

### 数据归属
- 数据与你的账户绑定
- 其他用户无法查看你的进度
- 你可以随时重置自己的进度

### 练习建议
1. **定期练习**: 进度会保存，不用担心中断
2. **适度重置**: 觉得需要重新开始时可使用重置功能
3. **查看统计**: 关注准确率和 WPM 的提升

### 进度管理
- 每次练习的进度独立保存
- 不同文件的进度互不影响
- 可以同时在多个文件间切换练习
