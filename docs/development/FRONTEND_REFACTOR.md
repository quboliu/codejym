## 🚀 快速开始

### 前置要求
- Node.js 16+
- npm 或 pnpm

### 安装依赖

```bash
cd frontend
npm install
```

这将安装：
- `vue@^3.4.0` - Vue 3框架
- `lucide-vue-next@^0.460.0` - 图标库（已添加到 package.json）

### 开发模式

```bash
npm run dev
```

访问 `http://localhost:5173` (或 Vite 显示的端口)

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `frontend/dist/` 目录。

## 🎯 设计哲学
